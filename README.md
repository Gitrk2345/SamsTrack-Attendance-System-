The SamStrack Attendance System is a Java Full Stack Web Application that digitizes and streamlines attendance tracking. The system allows administrators and users to manage attendance records effectively through secure login, intuitive UI, and robust backend APIs.

It demonstrates CRUD (Create, Read, Update, Delete) operations for users, attendance records, and reports, making it a complete end-to-end full stack project.

This project uses:

Frontend: React.js for building an interactive, responsive user interface.

Backend: Spring Boot REST APIs for business logic and CRUD operations.

Database: MySQL for structured data storage and persistence.
